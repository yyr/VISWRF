* About
/vis/ualize /wrf/ model out put using ncl.

* Dependencies
* how to use
- set VISWRF environmental variable in bashrc
- go to results directory
- run $VISWRF/conf.sh to create ~conf.ncl~
- edit ~conf.ncl~ if necessary
- call the scripts one

#+name: demo
#+BEGIN_SRC sh
  ncl $VISWRF/XXX.ncl
#+END_SRC
